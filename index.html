what is the valclass framework?

Valclass is the latest css framework (released in 2021). It comes with some of the latest features that have never been available in any css framework. We came with the aim to help web creators / developers to simplify and speed up their work process without burdening the web with a lot of code. Use our various features and build an amazing website made by yourself or your team. Make changes and stick with it until your design is always remembered

Main feature

v-directive

v-directive adalah cri khas dari valclass yang menjadi pembuka disemua class nya.

v-office

v-office is a class that makes developers an instant button with the export feature of various office files

v-increment

v-increment is a class that automatically creates a series of numbers 1 - ending

v-sosmed

v-sosmed is a class that creates an instant button with a feature to a specific social media link

v-action

v-action is a class that gives an animated touch when touched and clicked

v-result

v-result is a class that can add up between the first and second variables to produce a total

v-time

v-time is a class that can print the current time on your browser screen

v-reload

v-reload is a class that creates an instant button which when clicked reloads the browser

v-back-to-top

v-back-to-top is a class that allows users to go directly to the bottom of a web page

look forward to various other features

Responsive meta tag


Valclass was also developed to ensure that your web interface is user friendly and better. To ensure proper rendering and touch zoom for all devices, add a responsive viewport meta tag to your . To ensure proper rendering and touch zooming for all devices, add the responsive viewport meta tag to your <head>.

You can see an example of this in action in the starter template.


<meta name="viewport" content="width=device-width, maximum-scale=1, initial-scale=1" user-scalable=no>
